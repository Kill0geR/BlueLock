# BlueLock 🔐✨

**BlueLock** is a futuristic AI-powered password manager with a sleek, glassmorphism UI.  

---

## How it works (theory)

- Uses **hotkeys** to interact:
  - `F3` → Record login credentials
  - `F4` → Scan the screen for login fields
- AI analyzes screenshots or UI elements to detect login forms
- Matches stored credentials with detected logins and autofills them
- All data is stored **locally and securely** in an encrypted vault
- Floating widget shows AI status and scan progress

> Note: This is the theoretical workflow. The app is still under development!  

---

## Features planned

- Glassmorphism UI with bluish neon highlights
- Animated floating widget for scan feedback
- Secure local vault for credentials
- AI-assisted autofill across apps and browsers
